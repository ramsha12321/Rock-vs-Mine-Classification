# 🪨 Rock vs Mine Classification using Logistic Regression

## 📌 Project Overview

This project implements a Machine Learning model to classify underwater objects as **Rock** or **Mine** using sonar signal data. The model is built using **Logistic Regression**, a supervised learning algorithm for binary classification.

---

## 🎯 Objective

The objective of this project is to predict whether an object detected by sonar is:

- 🪨 Rock (R)
- 💣 Mine (M)

---

## 📊 Dataset

- Dataset: Sonar Dataset
- Features: 60 numerical features
- Classes:
  - Rock (R)
  - Mine (M)

---

## 🛠 Technologies Used

- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib
- Jupyter Notebook

---

## 🤖 Machine Learning Algorithm

- Logistic Regression

---

## 📂 Project Workflow

1. Import Libraries
2. Load Dataset
3. Data Preprocessing
4. Train-Test Split
5. Train Logistic Regression Model
6. Evaluate Model Performance
7. Make Predictions

## 🚀 How to Run

1. Clone this repository.
2. Install the required libraries:

```bash
pip install -r requirements.txt
```

3. Open the notebook:

```text
Rock and Mine Prediction.ipynb
```

4. Run all cells.
